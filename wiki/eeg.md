# Electroencephalography

Electroencephalography (EEG) measures electrical potentials generated by neuronal activity at the scalp surface. EEG signals result from synchronized postsynaptic potentials in cortical pyramidal neurons, primarily from apical dendrites oriented perpendicular to the cortical surface.

## Physics

EEG measures the electrical potential difference between electrodes on the scalp, arising from current dipoles in cortical tissue. The signal propagates through brain tissue, cerebrospinal fluid (CSF), skull, and scalp according to volume conduction principles governed by Poisson's equation:

```
∇ · (σ∇φ) = -I
```

where φ is the electrical potential, σ is tissue conductivity, and I represents neuronal current sources.

The forward model relating neural current density J to EEG measurements Y is:

```
Y = L·J + n
```

where L is the leadfield matrix encoding volume conduction physics and n is measurement noise.

## Tissue Properties

EEG signal propagation depends critically on tissue conductivity values:

| Tissue | Conductivity (S/m) | Source |
|--------|-------------------|---------|
| Gray matter | 0.33 | Gabriel et al. 1996 |
| White matter | 0.14 | Gabriel et al. 1996 |
| CSF | 1.79 | Gabriel et al. 1996 |
| Skull | 0.006-0.015 | Dannhauer et al. 2011 |
| Scalp | 0.43 | Gabriel et al. 1996 |

The skull's low conductivity (25-80× less than brain tissue) creates the primary impediment to EEG source localization, producing signal smearing and depth bias.

## Signal Characteristics

**Amplitude:** 10-100 μV at scalp (post-amplification)  
**Frequency range:** 0.5-100 Hz (clinical), up to 500 Hz (research)  
**Spatial resolution:** 1-3 cm due to volume conduction blurring  
**Temporal resolution:** 1 ms (limited by sampling rate, not physics)  

**Signal-to-noise ratio:** Varies by frequency band:
- Delta (0.5-4 Hz): highest SNR
- Theta (4-8 Hz): high SNR  
- Alpha (8-12 Hz): moderate SNR
- Beta/Gamma (>12 Hz): decreasing SNR with frequency

## Source Imaging Methods

NeuroJAX implements 15 EEG source imaging algorithms spanning five families:

### L2 Minimum Norm
- **MNE (Minimum Norm Estimate):** `neurojax.source.mne_inverse`
- **dSPM (Dynamic Statistical Parametric Mapping):** Noise-normalized MNE
- **sLORETA:** Standardized low-resolution electromagnetic tomography
- **eLORETA:** Exact LORETA with zero localization error

### Biophysical Priors
- **LAURA:** Local Autoregressive Average with 1/d³ spatial prior
- **VARETA:** Variable Resolution Electromagnetic Tomography

### Beamformers
- **LCMV (Linearly Constrained Minimum Variance):** Adaptive spatial filtering
- **SAM (Synthetic Aperture Magnetometry):** Pseudo-Z statistic beamformer
- **DICS:** Dynamic Imaging of Coherent Sources (frequency domain)

### Bayesian Sparse
- **CHAMPAGNE:** Convex Optimization Modeling for Sparse Learning
- **HIGGS:** Hierarchical Graphical Models for source connectivity

### Deep Learning
- **PI-GNN:** Physics-Informed Graph Neural Network on cortical mesh

## Head Modeling Pipeline

EEG forward modeling in neurojax follows this workflow:

1. **Anatomical segmentation:** CHARM 60-tissue classification from T1w
2. **Surface extraction:** Marching cubes for tissue boundaries  
3. **Conductivity assignment:** Literature values + qMRI-informed corrections
4. **Numerical methods:** BEM (OpenMEEG) or FEM (JAX-FEM) for leadfield computation
5. **Source space:** Cortical surface mesh from FreeSurfer or volumetric grid

### Differentiable Head Modeling

```python
from neurojax.geometry.fem_forward import assemble_stiffness, sigma_from_qmri

# Map qMRI parameters to conductivity
sigma = sigma_from_qmri(t1_values, bpf_values, labels, qmri_params)

# Assemble FEM system matrix
K = assemble_stiffness(vertices, elements, sigma)

# Gradient flows through entire forward model
grad_params = jax.grad(source_loss)(qmri_params)
```

## Preprocessing Pipeline

Standard EEG preprocessing in neurojax:

```python
from neurojax.preprocessing import asr_repair, ica_decompose, bandpass_filter

# Artifact Subspace Reconstruction
data_clean = asr_repair(raw_data, cutoff=20)

# Independent Component Analysis  
ica_weights, sources = ica_decompose(data_clean, n_components=32)

# Bandpass filtering
data_filtered = bandpass_filter(data_clean, l_freq=0.5, h_freq=40.0)
```

### ASR Parameters
- **Cutoff threshold:** 10-30 (standard deviations)
- **Window length:** 0.5 s 
- **Overlap:** 0.66
