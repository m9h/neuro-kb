# Hidden Markov Models for Brain Dynamics

Hidden Markov Models (HMMs) are probabilistic models for discovering discrete brain states from continuous neural time series. In neuroimaging, HMMs decompose M/EEG or fMRI data into a sequence of transient functional connectivity states with characteristic spectral signatures.

## Mathematical Framework

An HMM assumes observations Y(t) are generated by a latent discrete state sequence Z(t):

```
Z(t) | Z(t-1) ~ Categorical(π_{Z(t-1)})    [transition model]
Y(t) | Z(t) ~ N(μ_{Z(t)}, Σ_{Z(t)})        [observation model]
```

Where:
- π_k is the transition probability vector from state k
- μ_k, Σ_k are the mean and covariance for state k
- K is the number of states (typically 6-12 for MEG)

## Key Parameters

| Parameter | Typical Range | Description |
|-----------|---------------|-------------|
| K (states) | 6-12 | Number of discrete brain states |
| Window length | 200-500ms | Time window for covariance estimation |
| Regularization λ | 0.01-0.1 | Tikhonov regularization for covariance |
| Initialization | K-means | Clustering method for initial state assignments |

## Implementation Details

### Time-Delay Embedding (TDE)
Raw sensor data is typically time-delay embedded before HMM fitting:

```python
# neurojax implementation
X_tde = make_time_delay_embedding(X, delays=range(-2, 3))  # ±2 samples
gamma = hmm.fit(X_tde)  # State probabilities (T, K)
```

### State Characterization
Each state k is characterized by:
- **Mean activation** μ_k: spatial pattern of activity
- **Covariance** Σ_k: functional connectivity within the state  
- **Power spectral density**: frequency content of state activations
- **Fractional occupancy**: proportion of time spent in state k
- **State lifetime**: average duration before transitioning

## Inference Methods

### Forward-Backward Algorithm
Computes posterior state probabilities γ(t,k) = P(Z(t)=k | Y):

```
α(t,k) = P(Y(1:t), Z(t)=k)     [forward pass]
β(t,k) = P(Y(t+1:T) | Z(t)=k)   [backward pass]
γ(t,k) = α(t,k) × β(t,k) / P(Y)
```

### Viterbi Decoding  
Finds most likely state sequence:

```
δ(t,k) = max P(Z(1:t-1), Z(t)=k | Y(1:t))
ψ(t,k) = argmax δ(t-1,j) × π_{j,k}
```

## Validation Metrics

| Metric | Formula | Interpretation |
|--------|---------|----------------|
| Fractional occupancy | FO_k = Σ_t γ(t,k) / T | Time in state k |
| Mean lifetime | ML_k = Σ runs(k) length / n_runs | Average state duration |
| State switching rate | SR = Σ_t I[argmax γ(t) ≠ argmax γ(t+1)] / T | Transitions per second |
| Free energy | F = log P(Y\|Θ) - 0.5×p×log(T) | BIC model evidence |

Typical MEG values: FO ∈ [0.05, 0.3], ML ∈ [50ms, 200ms], SR ∈ [5, 15] Hz.

## Neuroimaging Applications

### Source-Space HMM
Applied to parcellated MEG/EEG source reconstructions:

```python
# Typical workflow
J = source_imaging(Y, L, method='LAURA')  # Source reconstruction  
J_parc = parcellate(J, atlas='Desikan')   # (T, 68) for Desikan atlas
gamma = hmm.fit(J_parc, K=8)              # State inference
```

### Spectral State Characterization
Each state exhibits characteristic power spectral densities:

```python
for k in range(K):
    state_data = J_parc[gamma[:, k] > 0.8]  # High-confidence state k data
    psd_k = welch(state_data, fs=fs)        # Power spectral density
    peak_freq[k] = freq[np.argmax(psd_k)]   # Dominant frequency
```

Common state types: alpha-rich (8-13 Hz), theta (4-8 Hz), beta (13-30 Hz), broadband high-activity states.

## Comparison with DyNeMo

| Aspect | HMM | DyNeMo |
|--------|-----|--------|
| States | Discrete | Continuous mixing |
| Transitions | Sharp switches | Smooth evolution |
| Covariance | Full per-state | Sparse time-varying |
| Computation | EM algorithm | Variational inference |
| Interpretability | High (discrete states) | Moderate (mode mixing) |
