# Magnetoencephalography

Magnetoencephalography (MEG) measures the magnetic fields generated by intracranial current dipoles, primarily from synchronous postsynaptic potentials in pyramidal neurons. MEG provides millisecond temporal resolution and 2-5 mm spatial resolution after source reconstruction.

## Physics

MEG detects the magnetic component of neural electromagnetic activity via the Biot-Savart law:

```
B(r) = (μ₀/4π) ∫ (J(r') × (r - r')) / |r - r'|³ d³r'
```

where `J(r')` is the primary current density in the brain and `B(r)` is the magnetic field at sensor position `r`.

Only **tangential** current sources relative to the skull surface generate detectable magnetic fields outside the head. Radial sources (perpendicular to cortical surface in sulci) are largely invisible to MEG, creating complementary sensitivity to EEG.

## Sensor Technology

| Parameter | Typical Value | Notes |
|-----------|---------------|-------|
| Sensor type | SQUID (Superconducting Quantum Interference Device) | Requires liquid helium cooling |
| Number of channels | 275-306 | CTF systems: 275ch, Neuromag: 306ch |
| Noise floor | 3-5 fT/√Hz | At 10 Hz |
| Bandwidth | DC - 1000 Hz | Limited by anti-aliasing filters |
| Sampling rate | 1000-2000 Hz | Nyquist for gamma oscillations |
| Sensor spacing | ~2-3 cm | Covers whole head |

Modern systems use **gradiometers** (measure spatial derivatives ∂B/∂x, ∂B/∂y) to suppress distant noise sources and improve spatial resolution over magnetometers.

## Forward Model

The MEG forward problem relates source current density J to measured magnetic field B:

```
B = L · J + n
```

where:
- `B` ∈ ℝ^(C×T) is the sensor data (C channels, T timepoints)
- `L` ∈ ℝ^(C×S) is the leadfield matrix
- `J` ∈ ℝ^(S×T) is the source current density (S source locations)
- `n` is additive sensor noise

The leadfield `L[c,s]` represents the magnetic field at sensor `c` generated by a unit dipole at source location `s`. Computation requires:

1. **Head model**: 3-shell BEM (scalp, skull, brain) or detailed FEM
2. **Source space**: Cortical surface mesh (~5000-10000 vertices)
3. **Sensor positions**: Digitized helmet geometry

## Source Reconstruction

MEG inverse problem is severely underdetermined (275 sensors, ~8000 sources). Multiple algorithms provide different bias-variance tradeoffs:

### Linear Methods
- **MNE (Minimum Norm Estimation)**: L2 minimum `||J||₂` with Tikhonov regularization
- **dSPM**: MNE with noise normalization `J_dSPM = J_MNE / sqrt(diag(Resolution))`
- **sLORETA**: Zero-localization error for point sources
- **eLORETA**: Exact localization via leadfield normalization

### Spatial Filters
- **LCMV (Linearly Constrained Minimum Variance)**: Minimizes output power while preserving signal from location of interest
- **SAM (Synthetic Aperture Magnetometry)**: Adaptive LCMV beamformer
- **DICS (Dynamic Imaging of Coherent Sources)**: Frequency-domain beamformer for oscillatory sources

### Bayesian Methods
- **CHAMPAGNE**: Sparse Bayesian learning with automatic relevance determination
- **VARETA**: Adaptive resolution based on 1/f³ spatial prior
- **HIGGS**: Hidden Gaussian graphical models for joint source-connectivity estimation

## Properties

| Property | Value | Source |
|----------|--------|---------|
| Temporal resolution | ~1 ms | Direct measurement of neural activity |
| Spatial resolution (sensor) | ~2-3 cm | Limited by sensor spacing |
| Spatial resolution (source) | 2-5 mm | After inverse modeling |
| Depth sensitivity | Surface-weighted | Tangential cortical sources only |
| Frequency range | DC - 500 Hz | Primarily 1-100 Hz for neural signals |
| Signal-to-noise ratio | 0.1-1 | Varies with source depth and orientation |

## Preprocessing Pipeline

1. **Noise suppression**: Temporal Signal Space Separation (tSSS) removes external interference
2. **Bad channel detection**: Automated based on variance and correlation
3. **Filtering**: High-pass (0.1-1 Hz), low-pass (40-100 Hz), notch (50/60 Hz powerline)
4. **Artifact removal**: ICA for eye blinks, heartbeat, muscle artifacts
5. **Epoching**: Extract trials time-locked to experimental events
6. **Baseline correction**: Remove pre-stimulus mean

## Head Model Requirements

MEG forward modeling requires accurate sensor positions and head geometry:

- **Sensor positions**: Digitized helmet coordinates (fiducials + head shape)
- **Anatomical MRI**: T1-weighted structural image for cortical surface
- **Coregistration**: Align MEG coordinate system with MRI
- **Boundary Element Method**: 3-shell model (brain, skull, scalp surfaces)
- **Source space**: Cortical surface mesh with ~4-8 mm vertex spacing

Unlike EEG, MEG does not require accurate skull conductivity values since magnetic fields pass through tissue unattenuated.

## Advantages over EEG

- **Reference-free**: Magnetic fields have no reference electrode artifacts
- **Less volume conduction**: Sharper spatial resolution, less signal spreading
- **Insensitive to skull**: No conductivity modeling uncertainties
- **Better deep source localization**: For tangentially-oriented sources
- **Higher temporal resolution**: No RC time constants from electrode impedances

## Limitations

- **Expensive infrastructure**: Liquid helium, magnetically shielded room
- **Radial source blindness**: Misses sources in sulcal walls
- **Limited portability**: Fixed installation in shielded environment  
- **Subject movement**: Rigid head position required during recording
- **Depth bias**: Exponential sensitivity drop with source depth
